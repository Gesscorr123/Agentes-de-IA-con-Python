MODELO DE PYTHON PARA GUARDAR MEMORIA A LARGO PLAZO
Descripción

Proyecto enfocado en el desarrollo de un agente de Inteligencia Artificial en Python con arquitectura de memoria híbrida (corto y largo plazo). Permite almacenar información relevante del usuario en PostgreSQL y mantener continuidad conversacional entre sesiones.

Desarrollado como parte del proceso de aprendizaje en la plataforma Ringa-Tech, con enfoque en construcción de agentes empresariales escalables.

Características

Memoria de corto plazo con buffer dinámico.

Persistencia de memoria en PostgreSQL (Supabase).

Function Calling con modelos LLM.

Integración con Google Calendar API.

Arquitectura modular y escalable.

Control de almacenamiento de información sensible.

Stack Tecnológico

Python

PostgreSQL

Supabase

Groq API

Google Calendar API
